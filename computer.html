Lesson 1: What is a Computer?

A computer is an electronic device that processes data and performs tasks based on instructions. It consists of hardware (physical parts) and software (programs).

Lesson 2: Basic Parts of a Computer

A computer has several important parts:

  • Monitor – Displays the output
  • Keyboard – Used for typing
  • Mouse – Used for navigation
  • CPU – The brain of the computer

Lesson 3: How to Use a Computer?

To use a computer, you need to switch it on, log in, and start working with applications like Word, Excel, or a web browser. Practice is the key to learning!
